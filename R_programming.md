# R Programming

### Data structures

**rep**

```r
c(rep(5, times = 3), rep(6, times = 4)
```

**seq**

```r
seq(1, 10, by = 2)  # returns c(1, 3, 5, 7, 9)

seq(1, 5, by = 1)  # returns c(1, 2, 3, 4, 5)
```

**vector**

Create a default vector

```r
vector(mode = 'numeric', length = 5)  # returns c(0, 0, 0, 0, 0)
```
